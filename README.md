A full-stack **Insurance Claims Management Platform** built using modern technologies like Next.js, NestJS, PostgreSQL, and cloud deployment.  
This application allows users to submit and track insurance claims while admins can manage and review them.

## 📌 Project Overview

This project simulates a **real-world insurance workflow system** with authentication and role-based access.

### 👤 Customer
- Sign up and login
- Submit insurance claims
- Track claim status
- View claim history

### 🛠 Admin
- View all claims
- Approve / Reject claims
- Monitor workflow and activity

---

## ✨ Features

- 🔐 JWT Authentication (Login & Signup)
- 🧑‍💼 Role-Based Access (Customer / Admin)
- 📊 Dashboard UI with analytics cards
- 📄 Claim submission & tracking
- ⚡ Real-time API integration
- 🎨 Modern UI with Tailwind CSS
- ☁️ Fully deployed (Frontend + Backend + DB)
